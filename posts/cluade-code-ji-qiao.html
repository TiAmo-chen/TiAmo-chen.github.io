Claude Code 隐藏命令与效率技巧（Obsidian 笔记）

来源：基于一篇实战经验整理，涵盖 /btw、/rewind、/insights、/model opusplan、/simplify、/branch、/loop、/remote-control、/export 及常用快捷键。

🚀 工作流与高级辅助

/btw — 临时插话，不污染上下文

  • 用法：在 Claude 执行长任务时，输入 /btw 然后提问。
  • 效果：回答不会中断原任务，也不加入对话历史，几乎不消耗额外 token。
  • 适用场景：突然想问“测试文件在哪个目录？”这类无关主任务的问题。

/rewind（或双击 Esc）— 智能回退

  • 用法：输入 /rewind 或按两下 Esc，弹出菜单选择回退范围。
  • 可选操作：
    • 回退代码和对话
    • 回退对话但保留代码
    • 回退代码但保留对话
    • 从该点开始压缩对话（释放上下文窗口）
  • 适用场景：实验不同方案，失败后保留对话记忆，直接换方向。

📊 分析与效率提升

/insights — 使用习惯分析报告

  • 用法：输入 /insights，自动生成一份本地 HTML 报告。
  • 内容：分析过去一个月你最常用的命令、重复操作模式、翻车现场，并推荐自定义命令和 Skills。
  • 建议：每月运行一次，重新认识自己的使用习惯。

🧠 模型与代码质量

/model opusplan — 混合模型规划执行

  • 用法：直接输入 /model opusplan 启用。
  • 原理：复杂推理时自动用 Claude Opus 4.6（规划），执行时切换到 Claude Sonnet 4.6。
  • 价值：对 Pro 订阅用户尤其友好，节省 Opus 额度，同时保证规划质量。

/simplify — 三合一代码审查

  • 用法：输入 /simplify。
  • 机制：同时启动三个 Agent，分别从 代码复用、代码质量、运行效率 三个角度审查你的改动。
  • 习惯建议：长对话或完成大功能后顺手运行一次，清理冗余 import、重复逻辑等。

🔄 会话控制与分支

/branch（原 /fork）— 分叉对话

  • 用法：输入 /branch，创建当前对话的分支（新会话），原会话不受影响。
  • 与 /rewind 区别：/rewind 是回退（后悔药），/branch 是分叉（平行宇宙）。
  • 适用场景：想同时尝试两种实现方式，保留原进度。

/compact — 手动压缩上下文

  • 用法：输入 /compact。
  • 作用：手动压缩对话历史，释放上下文窗口空间，避免因过长上下文导致回复质量下降。

⏱️ 自动化与远程控制

/loop — 定时重复任务

  • 用法：/loop <时间间隔> <任务描述>，例如 /loop 5m 检查部署状态。
  • 默认：间隔 10 分钟。
  • 生命周期：创建 3 天后自动过期并自我删除（最后一次触发后）。若需永久循环，建议使用桌面版。

/remote-control（或 /rc）— 手机遥控终端

  • 用法：输入 /rc 或 /remote-control，生成一个 URL。
  • 效果：手机浏览器打开该 URL，即可远程操作本地 Claude Code 会话。代码和文件始终留在本地，仅传输指令。
  • 特点：终端与手机实时同步，可交替使用。

💾 导出与历史管理

/export — 导出对话为 Markdown

  • 用法：输入 /export，当前整段对话导出为 .md 文件。
  • 用途：保存架构讨论过程、作为后续上下文、或跨 AI 工具协作（例如给 Codex 看）。

/clear — 重置对话上下文

  • 用法：输入 /clear。
  • 作用：清空当前会话历史，开启全新上下文。

⌨️ 快捷键（终端内）

快捷键（Mac 注意） 功能
Ctrl+V（不是 Cmd+V） 直接粘贴截图（Claude 可看图）
Ctrl+J 或 Option+Enter 换行（编写多行 Prompt）
Ctrl+R 搜索输入历史（快速复用之前的 Prompt）
Ctrl+U 删除整行输入
Shift+Tab 切换 Plan/Auto/Normal 模式

💡 实用建议

  1. 紧跟更新：Claude Code 更新极快，官方 CHANGELOG 和开发团队的 Twitter 是获取新功能的第一手渠道。
    🔗 官方更新日志

  2. 形成个人习惯：

    • 长任务中用 /btw 插问临时问题。
    • 完成大功能后运行 /simplify 审查代码。
    • 需要实验时用 /branch 分叉会话。
    • 每月跑一次 /insights 优化工作流。

  3. 安全提示：/remote-control 不暴露本地文件，放心使用。
